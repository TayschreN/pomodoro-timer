# Pomodoro Timer

Timer Pomodoro com catálogo de temas personalizáveis.

## Funcionalidades

- Timer Pomodoro com ciclos de Foco, Pausa Curta e Pausa Longa
- Catálogo de temas com cores personalizáveis
- Editor de temas com preview visual
- Configuração de durações para cada ciclo
- Notificação sonora e visual ao final de cada sessão
- Círculo de progresso animado
- Temas persistentes salvos no navegador

## Como usar

1. Abra o arquivo `index.html` no navegador
2. Escolha um tema no painel lateral
3. Ajuste as durações nas configurações (⚙)
4. Inicie o timer
